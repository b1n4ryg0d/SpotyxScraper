# SpotyxScraper

## Русский

Telegram бот для парсинга сайтов. Получает ссылку от пользователя и возвращает собранные данные.

### Команды
- `/start` - запуск бота, главное меню

### Установка

1. Установка библиотек:
```bash
python -m pip install -r requirements.txt
```
2. Заполнение .env файла:
BOT_TOKEN=ваш_токен_бота

3. Запуск бота:
```bash
python main.py
```

## English

Telegram bot for web scraping. Get a link from user and returns data.

### Commands
- `/start` - starting bot, main menu

### Installing

1. Installing libraries:
```bash
python -m pip install -r requirements.txt
```
2. Filling .env file:
BOT_TOKEN=your_bot_token

3. Bot running:
```bash
python main.py
```
